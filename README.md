# 📄 Мобільні рахунки

> **PWA додаток для генерації рахунків-фактур**  
> Працює офлайн, зберігає дані в IndexedDB, підтримує експорт/імпорт бекапів та підпис у PDF

[![PWA](https://img.shields.io/badge/PWA-Enabled-5A0FC8?style=flat-square&logo=pwa)](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps)
[![Made with JavaScript](https://img.shields.io/badge/Made%20with-JavaScript-F7DF1E?style=flat-square&logo=javascript)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

---

## ✨ Функціонал

| Функція | Опис |
|:--------|:-----|
| 📦 **Склад** | Додавання, редагування, видалення товарів. Відстеження кількості. |
| 👥 **База покупців** | Додавання, редагування, видалення покупців. Автопідстановка в рахунку. |
| 👤 **Профіль продавця** | Редагування реквізитів (ФОП, адреса, IBAN, банк). Додавання підпису. |
| 🧾 **Рахунки** | Створення, редагування, видалення. Автонумерація. Списання/повернення товарів. |
| 📄 **PDF генерація** | Рахунок у форматі PDF з підписом, таблицею, сумою прописом. |
| 💾 **Бекап** | Експорт/імпорт даних (JSON). Очищення всіх даних (з підтвердженням). |
| 📱 **PWA** | Встановлення на телефон як нативний додаток. Робота офлайн. |

## 🗂️ Детальний опис файлів

### **index.html**
- Розмітка сторінок: Головна, Склад, База, Профіль
- Нижня навігація (4 вкладки)
- Модальні вікна (рахунок, товар, покупець)
- Підключення бібліотек: `html2canvas`, `jsPDF`

### **css/style.css**
- Стилі карток, форм, кнопок, таблиць, модальних вікон
- Адаптивність під мобільні пристрої
- Нижня навігація
- Кастомний скролбар (прихований на телефоні)

### **js/app.js**
- Ініціалізація додатку (DB, завантаження даних)
- Навігація між сторінками
- Обробники подій (кнопки, пошук, модальні вікна)
- Реєстрація Service Worker (PWA)
- Обробники підпису (вибір файлу, видалення, прев'ю)

### **js/db.js**
- Відкриття IndexedDB (`InvoicesDB`, версія 6)
- Object stores: `profile`, `stock`, `customers`, `invoices`, `signature`
- Функції CRUD для профілю, складу, покупців, рахунків
- Оновлення складу після рахунку: `updateStockAfterInvoice`
- Функції для підпису: `saveSignatureFile`, `loadSignature`, `deleteSignature`

### **js/ui.js**
- Відображення списку товарів на складі (з пошуком)
- Відображення списку покупців (з пошуком)
- Відображення останніх рахунків на головній
- Оновлення статистики (кількість товарів, покупців, рахунків)

### **js/invoice.js**
- Створення нового рахунку (модальне вікно)
- Додавання/видалення товарів в рахунку
- Розрахунок сум (оригінал, знижка, зі знижкою)
- Автоматичне визначення наступного номера рахунку
- Збереження, редагування, видалення рахунків
- Списання/повернення товарів зі складу

### **js/pdf.js**
- Генерація PDF через `html2canvas` + `jsPDF`
- Два стовпчики: Постачальник / Одержувач
- Таблиця товарів (сірий заголовок)
- Підсумки: Разом, Знижка, Всього
- Сума прописом (українською)
- Підпис відправника (зображення з бази)
- Нижній колонтитул (дата створення)

### **js/modals.js**
- Модальне вікно додавання/редагування товару
- Модальне вікно додавання/редагування покупця

### **js/backup.js**
- Експорт окремих типів даних у JSON
- Імпорт з авто-визначенням типу
- Повна заміна даних при імпорті
- Очищення всіх даних (з подвійним підтвердженням)

### **js/utils.js**
- `escapeHtml` — захист від XSS
- `numberToWords` — перетворення числа в пропис (укр.)
- `showLoading`, `hideLoading` — індикатор завантаження

### **manifest.json**
- Назва: "Мобільні рахунки" / "Рахунки"
- Кольори: рожевий (#f06292), фон (#fef7f0)
- Іконки: 48, 72, 96, 144, 192, 512px
- Режим: standalone

### **sw.js**
- Кешування файлів для офлайн роботи
- Версія кешу: `invoice-app-v3`
- Автоматичне оновлення кешу

## 🚀 Встановлення на телефон

1. **Завантажте проект на GitHub Pages** або будь-який HTTPS хостинг
2. **Відкрийте посилання в Chrome** на телефоні
3. **Натисніть три крапки** → **"Додати на головний екран"**
4. **Підтвердіть** → додаток з'явиться з іконкою
